Class: Branching

This element describes the branching in a workflow from a source (start) structural element to 2 or more target structural elements, over a Transition element.

URI: odm:Branching

erDiagram
Branching {
    oid OID  
    nameType name  
    BranchingType type  
}
DefaultTransition {

}
Transition {
    oid OID  
    nameType name  
    string sourceOID  
    string targetOID  
}
TargetTransition {

}
ConditionDef {
    oid OID  
    nameType name  
}

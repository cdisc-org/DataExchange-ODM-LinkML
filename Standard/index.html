Class: Standard

The Standard element describes each standard used within the MetaDataVersion element.

URI: odm:Standard

erDiagram
Standard {
    oid oID  
    StandardName name  
    StandardType type  
    StandardPublishingSet publishingSet  
    text version  
    StandardStatus status  
    oidref commentOID  
}

Slots

Name Cardinality* and Range Description Inheritance
